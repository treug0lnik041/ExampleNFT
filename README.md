# Example NFT

This project demonstrates a basics of NFT (ERC721 standart).

## Clone
```shell
git clone https://github.com/treug0lnik041/ExampleNFT.git
npm i
```

## Compile contract

```shell
npx hardhat compile
```

## Test

```shell
npx hardhar test
```

## Deploy

```shell
npx hardhat deploy
```
